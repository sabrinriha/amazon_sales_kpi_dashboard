# amazon_sales_kpi_dashboard

An interactive Power BI dashboard that provides comprehensive insights into Amazon sales performance across multiple dimensions. The dashboard tracks key metrics including total sales of $12.64M, total profit of $1.47M, profit margin of 11.61%, and average order value of $491.39 across 26K orders.

Features interactive visualizations for sales and profit trends over time (2012-2015), geographic performance analysis through regional mapping, and detailed category breakdowns showing the distribution across Technology, Furniture, and Office Supplies. The dashboard includes analysis of top-performing products and customers, as well as sub-category performance metrics.

Users can filter data by country, region, city, category, and sub-category to drill down into specific business segments. The year selector allows for easy temporal analysis and year-over-year comparisons to identify trends and patterns in sales performance.
